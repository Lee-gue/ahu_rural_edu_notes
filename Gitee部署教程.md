# Gitee Pages 网站部署详细教程

> 本教程将教你如何免费部署「安大乡村教育笔记」网站

---

## 一、注册 Gitee 账号

### 1.1 访问 Gitee 官网
- 打开浏览器，访问：**https://gitee.com**

### 1.2 点击注册
- 点击右上角「注册」按钮

### 1.3 填写注册信息
- **姓名**：你的用户名（英文或拼音，如 litao2026）
- **邮箱**：常用邮箱
- **密码**：设置密码
- **手机号**：用于验证

### 1.4 完成验证
- 输入手机验证码
- 点击「立即注册」

---

## 二、创建代码仓库

### 2.1 进入创建页面
- 登录后，点击右上角「**+**」号
- 选择「新建仓库」

### 2.2 填写仓库信息

| 字段 | 填写内容 |
|-----|---------|
| **仓库名称** | `rural-edu`（或 `ahu-edu`、`xedu` 等你喜欢的名字） |
| **路径** | 自动生成，不用改 |
| **仓库介绍** | 安徽大学乡村教育笔记项目官网 |
| **是否开源** | 选择「公开」（必须公开才能免费使用 Pages） |
| **初始化仓库** | ✅ 勾选「使用 Readme 文件初始化这个仓库」 |
| **添加 .gitignore** | 不勾选 |
| **添加开源许可证** | 不勾选 |

### 2.3 点击创建
- 点击绿色按钮「创建」

---

## 三、上传网站文件

### 3.1 进入仓库
- 创建完成后，自动跳转到仓库页面

### 3.2 上传文件
- 点击页面上方的「**上传文件**」按钮

### 3.3 选择文件
- 点击「点击选择文件」或直接拖拽文件
- 选择以下两个文件：
  - `index.html`（主网站）
  - `admin.html`（管理后台）

### 3.4 提交上传
- **提交信息**：填写「上传网站文件」
- 点击「提交」按钮

---

## 四、开启 Gitee Pages 服务

### 4.1 进入服务页面
- 在仓库页面，点击顶部菜单「**服务**」
- 选择「**Gitee Pages**」

### 4.2 配置 Pages
- **部署分支**：选择 `master` 或 `main`
- **部署目录**：选择 `/`（根目录，默认）
- **强制使用 HTTPS**：建议勾选

### 4.3 启动服务
- 点击「**启动**」或「**更新**」按钮
- 等待几秒钟，页面会刷新

### 4.4 获取网站地址
启动成功后，页面会显示你的网站地址：
```
https://你的用户名.gitee.io/仓库名/
```

例如：
```
https://litao2026.gitee.io/rural-edu/
```

---

## 五、访问你的网站

### 5.1 主网站
```
https://你的用户名.gitee.io/仓库名/index.html
```
或直接：
```
https://你的用户名.gitee.io/仓库名/
```

### 5.2 管理后台
```
https://你的用户名.gitee.io/仓库名/admin.html
```

## 六、后续更新网站

### 6.1 修改文件
1. 进入 Gitee 仓库
2. 点击要修改的文件（如 `index.html`）
3. 点击右上角「编辑」按钮
4. 修改代码
5. 点击「提交」

### 6.2 上传新文件
1. 点击「上传文件」
2. 拖入新文件
3. 提交

### 6.3 更新 Pages
- 修改文件后，Gitee Pages 会自动更新
- 如果没有更新，可以回到「服务 → Gitee Pages」点击「更新」按钮

## 七、常见问题

### Q1: 页面显示 404 怎么办？
- 检查文件名是否正确（`index.html`）
- 检查 Pages 是否已启动
- 等待几分钟再试

### Q2: 网站样式错乱怎么办？
- 检查网络是否正常
- 清除浏览器缓存重试

### Q3: 如何绑定自定义域名？
- Gitee Pages 免费版不支持自定义域名
- 如需自定义域名，可考虑 GitHub Pages 或购买服务器

### Q4: 投稿数据存在哪里？
- 当前使用浏览器 localStorage 存储
- 换浏览器/清缓存会丢失数据
- 如需永久存储，需要购买服务器搭建后端
